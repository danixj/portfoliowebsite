Surveillance Night Vision Mobile Robot

about the project

The Surveillance Night Vision Mobile Robot is a sophisticated robotic platform designed for surveillance applications, particularly in low-light conditions. This project combines cutting-edge hardware components with advanced software programming to create a versatile and reliable autonomous system

Hardware used:

  • Raspberry Pi 4B:The heart of the system, providing computational power and interfacing with various peripherals.
  • Teensy 3.2: Handling low-level control tasks and interfacing with sensors and actuators.
  • RP Lidar: A crucial sensor for environment perception and navigation, enabling the robot to map its surroundings and navigate autonomously
  • Night Vision Camera: Equipped with infrared capabilities, allowing the robot to see clearly in low-light or no-light conditions.
  • Mobile Base: The physical platform on which all components are mounted, providing mobility and stability.

Software stack

  • ROS (Robot Operating System): Utilized for building and controlling the robot's software architecture. ROS provides a framework for managing hardware abstraction, communication between components, and highlevel robot behaviors.
  • Python: The primary programming language used for developing ROS nodes and implementing various functionalities, including sensor data processing, navigation algorithms, and decision-making.
  • OpenCV: Employed for image processing tasks, such as object detection and tracking, leveraging the night vision camera's capabilities.
  • Custom Navigation Stack: Developed to integrate sensor data from RP Lidar and implement algorithms for localization, mapping, and path planning.

key features

  1. Autonomous Navigation: The robot can navigate autonomously in its environment, avoiding obstacles and following predefined paths.
  2. Night Vision Capabilities: Equipped with a night vision camera, the robot can operate effectively in low-light or no-light conditions, extending its surveillance capabilities.
  3. Real-time Surveillance:The robot continuously streams video feed to a remote monitoring station, providing real-time situational awareness.
  4. Remote Control Interface: An intuitive user interface allows operators to remotely control the robot's movements, adjust camera settings, and initiate surveillance tasks.
  5. Scalability: The modular design of the system enables easy integration of additional sensors or functionalities to adapt to different surveillance scenarios

achievements

  • Successfully integrated multiple hardware components into a cohesive robotic platform.
  • Developed custom software algorithms for autonomous navigation and surveillance tasks.
  • Demonstrated the capability to operate effectively in challenging low-light environments, enhancing surveillance capabilities.

future enhancements

  • Integration of machine learning algorithms for advanced object recognition and anomaly detection.
  • Implementation of advanced path planning algorithms to optimize navigation efficiency.
  • Deployment of multiple robots for coordinated surveillance missions in larger environments

conclusion

The Surveillance Night Vision Mobile Robot project showcases the integration of state-of-the-art hardware and software technologies to create a versatile and effective surveillance solution. With its autonomous navigation capabilities and night vision capabilities, the robot is well-suited for a wide range of surveillance applications, from security monitoring to search and rescue operations.
