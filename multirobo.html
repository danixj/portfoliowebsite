Multipurpose Autonomous mobile Robot

about the project

The Multipurpose Industrial Grade Autonomous Mobile Robot is a culmination of advanced engineering and robotics, aimed at revolutionizing industrial automation. This project combines mechanical design, electrical engineering, and software development to create a versatile and efficient robotic platform capable of performing various tasks in industrial settings

Hardware used:

  • Raspberry Pi 4B:The heart of the system, providing computational power and interfacing with various peripherals.
  • Teensy 4.1:Acts as the central control unit,coordinating movements and executing tasks with reliability
  • RP Lidar: A crucial sensor for environment perception and navigation, enabling the robot to map its surroundings and navigate autonomously
  • Oak D lite Camera:The OAK-D Lite camera is a compact, power-efficient version of the OAK-D, featuring lower resolution mono cameras (640x480) and an integrated BMI270 IMU in current batches, designed for spatial AI functionality in a smaller form factor
  • BMS(battery management system): Ensures safe operation and efficient management of the robot's power source
  • Motor Driver:Regulates motor speed and direction for precise control and maneuverability

Software stack

  • ROS (Robot Operating System): Utilized for building and controlling the robot's software architecture. ROS provides a framework for managing hardware abstraction, communication between components, and highlevel robot behaviors.
  • Python: The primary programming language used for developing ROS nodes and implementing various functionalities, including sensor data processing, navigation algorithms, and decision-making.
  • OpenCV: Employed for image processing tasks, such as object detection and tracking, leveraging the night vision camera's capabilities.
  • Custom Navigation Stack: Developed to integrate sensor data from RP Lidar and implement algorithms for localization, mapping, and path planning.

key features

  1. Autonomous Navigation: Capable of navigating autonomously, avoiding obstacles, and adapting to changing environments
  2. Multi-Purpose Functionality: Designed to perform a variety of tasks such as material handling, surveillance, and navigation.
  3. Safety Features: Incorporates emergency switch and robust mechanical design to ensure safe operation in industrial environments
  4. User Interface: Intuitive interface allows for remote control and monitoring of robot operations.
  5. Scalability: The modular design of the system enables easy integration of additional sensors or functionalities to adapt to different surveillance scenarios

achievements

  • Successful integration of complex hardware and software components into a cohesive robotic platform.
  • Development of custom path planning algorithm and supervisory control code.
  • Demonstration of autonomous navigation and task execution capabilities in real-world scenarios.

future enhancements

  • Manipulator Integration: Expand robot capabilities for performing complex tasks such as picking, packing, and sorting.
  • Advanced Path Planning:Implement more sophisticated algorithms for optimized navigation efficiency
  • Human-Robot Interface Enhancement: Develop intuitive interfaces for improved user experience and interaction.

conclusion

The Multipurpose Industrial Grade Autonomous Mobile Robot represents a significant advancement in industrial automation, offering versatility, efficiency, and safety in various applications. With its advanced features and planned enhancements, the robot is poised to revolutionize operations in industries such as manufacturing, logistics, and warehousing.
